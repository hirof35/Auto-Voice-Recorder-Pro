Auto Voice Recorder Proマイクからの入力を常に監視し、一定以上の音量を検知したときだけ自動で録音を開始・保存する、Java製のオートレコーダーです。会議の議事録作成、寝言の記録、楽器の練習など、無音時間を省いて効率よく録音したい場面に最適です。✨ 特徴無音カット録音: しきい値（感度）を設定し、必要な音がある時だけファイルに書き込みます。プリロール（先行録音）機能: 音を検知する約0.4秒前からのデータを保持しているため、言葉の冒頭が途切れるのを防ぎます。自動ファイル分割: 録音中に一定時間（約1.2秒）無音が続くと、自動でファイルを保存して再び待機状態に戻ります。ステレオ・高音質: 44.1kHz / 16bit / ステレオのWAV形式で保存します。リアルタイム・レベルメーター: 現在のマイク入力レベルを視覚的に確認できます。🛠 動作環境Java: JDK 17 以上OS: Windows, macOS, Linux (マイク入力が可能な環境)🖱 使い方監視開始: 「監視開始 / 停止」ボタンを押すと、マイクの監視が始まります（この時点ではまだ録音されません）。しきい値の設定: 「開始しきい値設定」スライダーを調整します。スライダーを左に動かすと、小さな音でも録音が始まります。スライダーを右に動かすと、大きな音にだけ反応します。録音: 音声がしきい値を超えると、ステータスが「録音中...」に変わり、ファイルへの書き込みが始まります。自動保存: しばらく無音が続くと、自動的に recordings フォルダ内にWAVファイルが生成されます。📂 保存場所と命名規則実行ファイルと同じディレクトリに recordings フォルダが作成されます。ファイル名形式: Rec_yyyyMMdd_HHmmss.wav⚙ 技術仕様項目設定値サンプリングレート44,100 Hzビット数16 bitチャンネル数2 (Stereo)エンコーディングPCM_SIGNED (Little Endian)先行バッファ約 0.4 秒📝 注意事項環境ノイズ: エアコンの音やPCのファンなどの環境ノイズが大きい場合、しきい値を適切に上げないと録音が止まらなくなることがあります。ファイルサイズ: 長時間の監視を行う場合は、ストレージの空き容量にご注意ください。ライセンスこのソフトウェアはオープンソース（MITライセンス）として公開されています。自由に変更・配布が可能です。
